# Cryptocurrencies


## Overview

Accountability Accounting, a prominent investment bank, is interested in offering a new cryptocurrency investment portfolio for its customers. This project focused on creating a report of what cryptocurrencies are on the trading market. Using unsupervised machine learning, the cryptocurrencies will be grouped to create a classification system for this new investment. The data required preprocessing and standardizing in order to prepare it for machine learning.
